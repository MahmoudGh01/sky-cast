# SkyCast

SkyCast is an Expo + TypeScript mobile weather app starter. It currently shows current weather and a short forecast for a sample location using the Open-Meteo API.

## Potential Features

- Use current location
- Search by city name
- Save favorite cities
- Compare weather across multiple cities
- Show historical weather trends

## Getting Started

```bash
npm install
npm run start
```

## Linting

```bash
npm run lint
```
